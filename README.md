dvdTaiki was made with [pygame](https://www.pygame.org/) to make you feel nostalgic.

## Getting Started
### Installation
1. Clone this repository from GitHub.
```
git clone https://github.com/miyakawa2000/dvdTaiki.git
```
2. Install pygame.
```
pip install pygame
```
### Run
1. Change the directory to dvdTaiki.
```
cd dvdTaiki/
```
2. Run the code.
```
python3 main.py
```
